# Healthcare_Disease_Analysis-Dashboard
Healthcare Disease Analysis Dashboard

**Project Overview**
This project analyzes healthcare data containing 20,000 patient records and 32 attributes to uncover patterns in disease distribution, hospitalization trends, and lifestyle risk factors. The objective is to transform raw healthcare data into meaningful insights using data analytics and visualization techniques.

**Tools and Technologies**
->Python (Pandas) – Data loading, cleaning, and preprocessing
->MySQL – Data storage and query-based analysis
->Microsoft Power BI – Interactive dashboard and data visualization
->Visual Studio Code – Development environment

**Project Workflow**
-Loaded the healthcare dataset using Python.
-Performed Exploratory Data Analysis (EDA) and cleaned the dataset.
-Stored the processed data in a MySQL database.
-Executed SQL queries to analyze healthcare patterns.
-Connected the database to Power BI and built an interactive dashboard for visualization.

**Key Insights**
#Identified the most common disease categories affecting patients.
#Analyzed state-wise distribution of disease cases.
#Examined the relationship between age groups and disease types.
#Studied the impact of smoking on disease occurrence.
#Observed seasonal trends in disease cases.
#Evaluated average hospitalization duration by disease category.

**Dashboard**
The Power BI dashboard presents multiple visualizations including bar charts, maps, line charts, and stacked charts to provide clear insights into healthcare patterns and trends.

**Conclusion**
This project demonstrates the use of data analytics tools to analyze healthcare data and generate meaningful insights that support better understanding of disease patterns and patient health trends.
